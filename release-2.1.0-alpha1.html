LibGD 2.1.0-alpha1 release

Wed 10 April 2013
By Ondřej Surý

In News.

GD team proudly announces that the 2.1.0-alpha1 version of GD Graphics Library has been released. Most of the improvements that PHP GD team had made to GD Graphics Library has been merged into 2.1.0-alpha1 release.

You can download the 2.1.0-alpha1 version of GD Graphics Library from the libgd project.

We would like to invite everyone to come and test the new library and report any issue found to our issue tracker.
